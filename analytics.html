Analytics Monitor for MT5

リアルタイムでトレード成績をダッシュボード表示。MFE/MAE計測・CSVログ出力にも対応。

無料ダウンロード 最新バージョン: v1.0 (MT5版)

FEATURES

📈

リアルタイム ダッシュボード

トレード数、勝率、プロフィットファクター(PF)、最大ドローダウンなどの重要指標をチャート上へ綺麗に可視化します。

🎯

MFE / MAE の自動追跡

エントリー後の最大順行幅（MFE）と最大逆行幅（MAE）を追跡計算。利確・損切りの最適化をサポートします。

📂

詳細 CSV ログの自動保存

日別の詳細トレード結果や月別サマリーデータを指定フォルダへ自動で集計・出力し、手軽に分析を行えます。

HOW TO INSTALL

01

ダウンロード

上のダウンロードボタンからZIPファイルを取得します。

02

フォルダを開く

MT5の「ファイル」メニューから「データフォルダを開く」を選択します。

03

配置する

「MQL5」＞「Experts」フォルダにファイルを格納します。

04

適用する

MT5のナビゲーターを右クリックして「更新」し、チャートへ適用します。
